# DWPS Greater Noida — Homepage Clone

A pixel-perfect Next.js 14 clone of the Delhi World Public School, Greater Noida homepage.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS with custom theme
- **Fonts**: Rubik + Marcellus (Google Fonts via next/font)
- **Icons**: React Icons
- **Animations**: CSS animations + Framer Motion ready

## Features

- ✅ Sticky top bar with contact info & social links
- ✅ Sticky header with navigation dropdowns & mobile hamburger menu
- ✅ Notice ticker/marquee with school announcements
- ✅ Hero banner with auto-advancing slider (3 slides)
- ✅ About section with image, key points, and features grid
- ✅ Highlights/Why Choose Us section with 8 feature cards
- ✅ Animated stats counter section
- ✅ Enrollment CTA with quick enquiry form
- ✅ Photo gallery with category filter and lightbox
- ✅ Testimonials carousel
- ✅ News & Events section with upcoming events sidebar
- ✅ Multi-column footer with quick links, contact info, social links
- ✅ Fully responsive (mobile, tablet, desktop)

## Color Scheme

- **Primary Green**: `#056a43`
- **Orange Accent**: `#ef7f1a`
- **Yellow**: `#ffcc00`
- **Background**: `#F6F6F1`

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm run start
```
